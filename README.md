cordova-sailfish
================

Cordova-sailfish is a SailfishOS port of the Apache Cordova project. 

#Requirements

[Saiflish OS SDK](https://sailfishos.org/)

#Supported APIs

* Accelerometer
* Compass
* Device
* Notification
* Connection
* Events
* in-app browser

# Work in progress

* Contacts
* Camera
* Geolocation
* Storage
